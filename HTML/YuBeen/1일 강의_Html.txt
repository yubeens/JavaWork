- HTML , CSS, 자바스크립트 사용

백엔드 개발
- 서버 / 사용자 뒤에서 보이지 않는 영역
- 데이터베이스를 설계하거나 데이터 처리
- 자바, PHP, 파이썬 등 프로그래밍 언어 사용
- 안에 들어갈 데이터를 주소창에 저장


비쥬얼 스튜디오
디버깅 실행 Ctrl + F5

head - 설정, 제목<title>
body - 본문

header nav - 로고나 검색창, 목록 등
main, contens - 메인내용 (섹션 등)
footer - 회사이름 고객센터 참고사항 등등

<h1> 대제목 - 크고 볼드체 (한 페이지에 1개만 사용)
(h1~h6 : 제목을 표시하는 태그)
 - h2~h6태그는 여러개를 중복해서 사용가능, 순서대로 사용할 것

(Ctrl + / ) 주석 : HTML파일을 실행 했을 때 코드로 인식하지 않는 부분 -> 코드에 대한 설명이나 주의사항

Alt + Shift + 아래화살표 : 한줄 복사하기

p태그 : 텍스트 단락, 텍스트 앞뒤로 빈줄이 생기면서 텍스트 단락이 만들어짐
br태그 : 줄을 바꾸는 태그, 엔터와 같은 기능
div태그 : 구역을 나누는 태그, 외관 꾸밀 때 (p태그의 기능과 비슷하나 css에서 구역을 나누기 위해 더 유용히 사용, )
pre태그 : 텍스트를 적혀있는 그대로 출력하는 태그 (공백표시 가능 , br태그 기능없이 엔터기능도 가능)
hr태그 : 단락을 나누는 가로줄 생성
blockquote태그 : 다른 텍스트보다 안으로 들여 써짐. 
strong, b 태그 : 볼드체 (Strong - 강조)
em, i태그 : 이탤릭체
<ol><li>태그 : 목록태그, 자동으로 리스트의 숫자를 매겨줌
★<ul><li>태그 : 비순서 목록태그, 리스트 앞에 동그라미
dl태그 dt(제목,이름), dd(값) : 설명 목록을 의미하는 태그


<table 테이블 태그>
caption - 표 제목
table - 표 전체
tr - 행
td -셀
th - 제목 셀

(눈에 띄지않음 , 정보만 입력)
thead - 제목
tbody - 본문
tfoot - 요약
